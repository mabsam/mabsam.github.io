RELATIVE, ABSOLUTE, AND FIXED POSTITIONING

While designing websites you need to control how elements behave and are displayed on the page. Let's say you wish to position elements next to each other or fix a certain element at the top of your page. That's when CSS's position property comes into play. Though it's mighty of doing much more and takes in five values: static, relative, absolute, fixed, and sticky, today we'll be focusing on the relative, absolute and fixed values.

Learning Competencies:

  • Compare relative, absolute and fixed positions to determine their differences.
  • Explain how to use these values.

POSITION: RELATIVE

Relative value allows you to position an element relative to its default static position on the page. However, if you just give an element position: relative it will not change its location. To move a relatively positioned element you need to tell it where to go by accompanying the position declaration with one or more of the following properties:

  • top - moves the element down from the top.
  • bottom - moves the element up from the bottom.
  • left - moves the element away from the left side (to the right).
  • right - moves the element away from the right side (to the left).

POSITION: ABSOLUTE

When an element has position: absolute, all other elements will act like it does not exist on the page. It makes position: absolute very useful when you want to stick an element in a specific position, but you do not want anything else to move around it.

The position of the absolute element is determined by its parent element, so the element will be located relative to its closest parent element. Top, bottom, left, right properties allow you to modify the final position of the element.

POSITION: FIXED

Fixed position is similar to absolute position since it removes the element from the document flow, but fixed position elements are always positioned relative to the screen no matter what position its parent elements are.

Fixed positioned elements stay on the screen when you scroll.
