JAVASCRIPT FUNDAMENTALS

An analogy to describe the differences between HTML and CSS

HTML is like a newbuild house - it has walls, floor, ceiling, doors, and windows, it also has a certain number of floors and rooms, however, living in this house would not be comfortable yet.

CSS is like design and styling of this house - adding some paintings here and there, comfy sofas, carpets, tables and chairs to make living in it more enjoyable.

JavaScript is like adding some interactive features to the house: plumbing system, electricity, gas.

Control flow and loops

Java Script control flow is the way the computer runs code from top to bottom, starting from the first line of code and ending at the last line, except for the situations when it gets a statement that changes the control flow of the program such as loops, conditionals etc.

Sometimes while coding you may find yourself in a situation where you need to perform certain tasks repeatedly. Loops give an opportunity to repeat a piece of code until a certain condition is met. It is like painting a wall. In order to achieve a rich and deep colour, you will have to cover the wall with two or even three coats of paint.

Mighty DOM

DOM stands for Document Object Model. JavaScript accesses the contents of a webpage making use of DOM which is created by the browser when a page is rendered. It is created in the system like a tree structure. The DOM lets you manipulate the elements and their contents, but first we need to reach the corresponding DOM object.

The main 'gateway' to DOM is the 'document' object, since all operations on the DOM start there. Interacting with a webpage is the same as interacting with its Document Object Model. Since a web browser creates DOM whenever a webpage gets rendered, you can add or remove HTML elements by manipulating the DOM itself. Also, you can style the elements using the CSS stylesheet, add user interaction by attaching event listeners, like click or a keypress on the keyboard, to the HTML elements.

For instance, the following command will get the first paragraph and assign it to a variable 'p':

let p = document.querySelector('p')

p.classList.add('blue') will give it a class 'blue'.

Accessing data from arrays and objects.

The properties of Objects can be accessed by using either dot notation or square bracket notation:

visitor {name: Josh, address: “1 Queen St”, phone: 222 222}

visitor.name or visitor[name]

Unlike objects, array contents can be accessed using their index position starting from 0. Arrays are accessed like this: shoppingList = [“pasta”, “rice”, “cheese”]; If you need to access “cheese” you would type shoppingList[2].

Functions

A function is a reusable block of code that groups together a sequence of statements to perform a particular task. JS function is executed when it is called.

function subtraction(number1, number2) { return number1 - number2; }

Calling the function:

subtraction(10, 5); // 5

Creating a function allows you to avoid writing the same code multiple times for different inputs.
